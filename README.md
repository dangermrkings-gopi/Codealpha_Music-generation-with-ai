# 🎵 AI Music Generator

A full-stack application that utilizes deep learning to compose music. This system allows users to train a model on MIDI datasets and generate new, unique musical sequences through an interactive, animated web interface.

## 🚀 Features
* **AI Engine:** Uses LSTM (Long Short-Term Memory) networks to analyze and replicate musical patterns.
* **MIDI Processing:** Integrates `music21` to parse, clean, and convert MIDI files into trainable data sequences.
* **Modern UI:** Features an animated, responsive frontend built with Tailwind CSS and Framer Motion.
* **Flask Backend:** A lightweight, production-ready Python API for model inference.
